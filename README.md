# AI上司と働く1週間

「あなたの上司はAIになる」という未来コンセプトを、被評価者目線で体験するためのストーリー型Webプロトタイプです。

## このプロトタイプの目的

半期ごとに思い出して評価される体験から、日々の行動・成果・学びが自然に記録され、必要な時に成長相談へつながる体験への変化を検証します。

実データ連携やAI API連携は行わず、架空の人物・架空の業務を使った「動く紙芝居」としてユーザーテストで利用できます。

## 体験の流れ

1. **Before：半期評価の世界** — 半年分の頑張りを後から思い出すモヤモヤを確認します。
2. **Concept：AI上司が日々伴走する世界** — AI上司が評価者ではなく成長コーチャーとして伴走する未来を提示します。
3. **Morning：朝のAI伴走** — 今日の予定、役割、成長テーマを一緒に確認します。
4. **Work：会議前相談・会議後記録** — 会議前の相談と、会議後の貢献記録を体験します。
5. **Support：作業中の迷いをAIへ相談する** — 過去の経緯を踏まえた選択肢とおすすめを確認します。
6. **Evening：仕事終わりの活動要約** — 1日の活動が成果・行動・学び・困りごとに整理されます。
7. **Growth：1週間後の成長データ** — 半期を待たずに現在地と次の成長テーマが見えます。
8. **Meeting：上司・AI・本人の三者面談** — 評価確認ではなく、成長相談として面談する未来を体験します。

## ローカルでの開き方

このリポジトリを取得した後、ブラウザで `index.html` を直接開いてください。

```bash
open index.html
```

Windowsの場合は、エクスプローラーで `index.html` をダブルクリックして開けます。

簡易サーバーで確認したい場合は、以下のコマンドも利用できます。

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## ユーザーテストで確認したいこと

- コンセプトに共感できるか
- AI上司に驚きがあるか
- 日々の記録・伴走に期待できるか
- 監視感よりも成長支援として受け止められるか
- 半期評価から日々の成長相談への変化が伝わるか

## 操作方法

- 画面下部の「次へ」「戻る」でストーリーを進めます。
- キーボードの左右矢印キーでも移動できます。
- 画面下部に現在のステップ番号と8つのステップインジケーターが表示されます。
- 最後の画面では「次へ」が「最初に戻る」に変わります。
